    ANSIBLE GALAXY IMPORT CHANGES

    Please be aware that there's a change made in Ansible Galaxy to require mandatory requires_ansible in the meta/runtime.yml file for uploading a collection (#122, #124). After you've added the collection, you will probably hit ansible-test failure in sanity checks if you use it with the --venv option, since it is unable to check this field due to the failing requirement. You will need to have the meta/runtime.yml file now for your collection to be uploaded.
